# Demo guide

Run:

```bash
uv run research-desk demo
```

The sample topic asks how autonomous multi-agent research systems should be
designed and evaluated. The expected visible trajectory is Gathering,
Organizing, Reviewing, and Writing. The completed artifact includes inline
`[S#]` citations, a source list, critic score, limitations, and export paths.

For the workbench, run `uv run research-desk ui`, enter the sample topic shown
above, keep `Bundled demo` selected, and start research. The source library labels
every bundled item `demo_fixture`.
This demonstration does not claim that the sources were retrieved live or that
the application is hosted in production.
